## Expo Router App — Scaffolding Documentation

### Steps Followed

#### Navigate to project directory

```
cd prodev-mobile-setup/prodev-mobile-app-0x00
```

#### Initialize Expo project

```
npx create-expo-app@latest .
```

#### Modify Home Screen

File updated:

```
app-example/app/(tabs)/index.tsx
```

Changed `Welcome!` ➜ `** First App Created**`

#### Run application

```
npx expo start
```

* iOS → scan QR with Camera app
* Android → scan QR with Expo Go

#### Reset project

```
npm run reset-project
```

### Reset Observation

* deletes node_modules
* clears cache
* reinstalls fresh dependencies
* restores initial Expo template state
